### Column Breakdown

- **Column A:** Checkbox — Task completion checkbox
- **Column B:** Task ID — Auto-generated unique identifier
- **Column C:** Task Name — Description of the task
- **Column D:** Category — Task category (Study, Networking, Admin, Personal, Project, Work, House Work)
- **Column E:** Priority — Task priority level (High, Medium, Low)
- **Column F:** Status — Task status (Not Started, In Progress, Completed)
- **Column G:** Start Date — Auto-populated when status changes to In Progress
- **Column H:** Due Date — Manual input, deadline for task
- **Column I:** Completion Date — Auto-populated when status changes to Completed
- **Column J:** Duration in Hours — Calculated from start to completion date
- **Column K:** Duration in Minutes — Calculated from start to completion date
- **Column L:** Days Overdue / Status Alert — Shows overdue status with colour coding
- **Column M:** Priority Flag — Escalates high priority tasks that are overdue or due soon
- **Column N:** Completion Rate — Running percentage of completed tasks
- **Column O:** Workload by Category — Shows task count and percentage per category
- **Column P:** Weekly Task Count — Counts tasks started in current week
- **Column Q:** Monthly Task Count — Counts tasks completed in current month
- **Column R:** Estimated vs Actual Time — Compares estimated hours to actual duration
- **Column S:** Task Efficiency Score — Rates task completion quality (0-100 scale)
- **Column T:** Overdue Count by Category — Shows overdue tasks per category
- **Column U:** Average Completion Time by Category — Average hours per category
- **Column V:** Task Velocity — Tasks completed per week
- **Column W:** Deadline Adherence Rate — Percentage of on-time completions
- **Column X:** Overall Productivity Score — Composite score out of 100
- **Column Y:** Estimated Time in Hours — Manual input for estimated task duration
- **Column Z:** Estimated Time in Minutes — Auto-converted from column Y
